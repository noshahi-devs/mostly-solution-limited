# Mostly Solutions Limited Website

This is a premium, professional Angular project for a car mechanic business.

## Getting Started

To run this project, you need to have **Node.js** and **NPM** installed on your machine.

### Installation

1. Open your terminal in this directory.
2. Install the dependencies:
   ```bash
   npm install
   ```

### Development Server

Run the following command to start the development server:
```bash
ng serve
```
Navigate to `http://localhost:4200/`. The app will automatically reload if you change any of the source files.

## Design Details
- **Branding**: Green & Blue color palette based on the "Mostly Solutions" logo.
- **Styling**: Vanilla CSS with glassmorphism effects and modern typography (Outfit).
- **Architecture**: Angular 17+ Standalone Components.
- **Assets**: includes custom-generated photorealistic images for hero and service sections.
